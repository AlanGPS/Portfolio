Os dados utilizados neste estudo foram captados através do Kaggle, no link:

https://www.kaggle.com/doaaalsenani/usa-cers-dataset?select=USA_cars_datasets.csv

Nesse conjunto de dados, são apresentados os carros usados a venda em todo o EUA, hospedados no site AUCTION EXPORT.com, dos anos de 1973 a 2020. Assim como algumas de suas características, como preço, ano de frabricação, condição de uso, cor, marca, modelo e milhas/kilometros rodadas.
